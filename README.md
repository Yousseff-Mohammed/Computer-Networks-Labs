# Computer-Networks-Labs

### Lab 1

Simple **TCP and UDP echo servers and clients** implemented in Python.

- UDP echo server and client
- TCP echo server and client
- The TCP server supports **multiple clients** using threads

### Lab 2

A **Python GUI email client** demonstrating **SMTP and IMAP communication**.

Features:

- Send emails using **SMTP**
- Retrieve emails using **IMAP**
- Graphical interface for interacting with the mail client
